# RTL-to-GDS Implementation of an ALU using OpenLane and SKY130

## Project Overview

This project demonstrates the complete RTL-to-GDS implementation of an Arithmetic Logic Unit (ALU) using the OpenLane ASIC design flow and the SKY130 Process Design Kit.

The goal of this project was to understand each stage of the digital ASIC design flow, starting from writing the RTL in Verilog and ending with the generation of the final GDSII layout.

Throughout this project, I explored the complete physical design flow including synthesis, floorplanning, placement, clock tree synthesis, routing, timing analysis, and signoff.

---

## Repository Contents

- **src/** – Verilog source files of the ALU
- **tb_alu_top.v** – Testbench used for functional verification
- **alu_top.vcd** – Simulation waveform generated during verification
- **reports/** – Reports generated during different stages of the OpenLane flow
- **runs/** – Final outputs generated by OpenLane, including layout files and implementation results
- **config.json** – Project configuration used by OpenLane
